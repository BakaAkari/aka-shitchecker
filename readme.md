# aka-shitchecker

对群成员进行随机数鉴定并执行禁言的 Koishi 插件。

## 功能

- 使用 `惩戒 @群成员` 指令对特定群成员进行随机数鉴定
- 根据鉴定结果执行禁言操作
- 仅管理员可使用

## 鉴定规则

### 1. 基础豁免检定（d20）

- **≥16**：成功豁免，无后续影响
- **≤15**：豁免失败，进入禁言时长判定阶段

### 2. 禁言时长判定（d20，仅在豁免失败时）

- **1**：大失败（72小时）
- **2-5**：1小时
- **6-15**：12小时
- **16-19**：24小时
- **20**：大成功（豁免禁言）

## 使用方法

1. 在配置中添加管理员用户ID到 `adminUsers` 列表
2. 在群组中使用 `惩戒 @群成员` 指令
3. 系统会自动进行两次 d20 检定，并根据结果执行禁言

## 配置

```json
{
  "adminUsers": ["管理员用户ID1", "管理员用户ID2"]
}
```

## 注意事项

- 仅支持 QQ 平台（onebot 适配器）
- 确保机器人有禁言权限
- 不能对自己或机器人使用惩戒指令
- 只能在群组中使用
